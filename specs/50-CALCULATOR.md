# 50 - Web Calculator

## 1. Goal
Build a high-quality web-based calculator for verification purposes.

## 2. Requirements
- **Format**: Single HTML file `calculator.html`.
- **UI**: Premium look using vanilla CSS.
- **Functions**: Addition, Subtraction, Multiplication, Division.
- **Edge Cases**: Division by zero must display "ERROR".

## 3. Verification
- `tests/contract/test_calculator.py` must exist and pass.
- `calculator.html` must be visually high-quality.
